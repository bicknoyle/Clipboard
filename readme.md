## Clipboard

Clipboard allows you to create online surveys. Built with [Laravel](http://laravel.com/)!

## Installation

    $ git clone https://github.com/bicknoyle/Clipboard.git
    $ cd Clipboard
    $ composer install
    $ php artisan migrate
